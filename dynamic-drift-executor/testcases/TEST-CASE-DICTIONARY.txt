TESTING RECOMMENDATIONS:
  1. Start with LOW severity tests (safe, tag-only changes)
  2. Then try MODERATE severity (requires stop, but reversible)
  3. Finally HIGH severity (in test environment only!)
  4. Never run HIGH-001 (instance termination) in production
  5. Some tests require placeholder values (SG IDs, Volume IDs, etc.)
